# RoleSync

## Bot

uses nodejs. make sure to install everything.

### Example config for the bot

```json
{
  "token": "discordbottoken",
  "prefix": "+",
  "port": 6000,
  "guild": "guildid",
  "channel": "botchannelid",
  "roles": [
    {
      "id": "adminroleid",
      "ign": "admin"
    }
  ]
}
```
